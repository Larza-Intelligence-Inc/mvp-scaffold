# Generated API clients

One subfolder per OpenAPI source. Types are generated from a **committed spec file**, never from a live URL.

| Folder | Spec source | Regenerate |
|--------|-------------|------------|
| `backend/` | [`../../backend/openapi.json`](../../backend/openapi.json) | `npm run generate:api` |

## Workflow after API changes

```bash
# 1. Emit the OpenAPI lockfile from the Hono app (no server required)
cd backend && npm run openapi:emit

# 2. Regenerate TypeScript types for openapi-fetch
cd ../frontend && npm run generate:api
```

Or use the Cursor skill **generate-frontend-sdk**, which runs both steps.

## `backend/` layout

- `schema.ts` — generated by `openapi-typescript` (do not edit)
- `client.ts` — hand-written `openapi-fetch` factory; injects `baseUrl` from env
